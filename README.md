Graph Representation Converter (C)

Simple C console application that converts between the three main graph representations:
	•	Incidence Matrix
	•	Adjacency Matrix
	•	Adjacency List

The program reads one representation from the user and automatically generates the other two.

Features
	•	Supports directed graphs
	•	Supports self-loops
	•	Dynamic memory allocation (malloc, free)
	•	Full conversion between all representations
